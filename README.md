# 📊 Site Agenda — Consultoria de Crédito Planejado

Landing page desenvolvida para captação de clientes interessados em **crédito planejado (imóvel e veículo)** com simulação simples e envio direto via WhatsApp.

---

## 🚀 Sobre o projeto

O **Site Agenda** é uma página web responsiva criada para:

- Simular crédito de forma rápida
- Facilitar o contato direto com o cliente via WhatsApp
- Apresentar o serviço de forma clara e profissional
- Gerar leads qualificados

## 🧠 Como funciona

### 1. Simulação

O usuário preenche:

- Nome
- WhatsApp
- Tipo de crédito
- Valor desejado
- Prazo

---

### 2. Cálculo

O sistema calcula automaticamente a parcela com base em:

| Tipo       | Taxa aplicada |
|-----------|-------------|
| Imóvel    | 24.2%       |
| Veículo   | 16.2%       |

**Fórmula:**

```

total = valor + (valor * taxa)
parcela = total / prazo

````

---

### 3. Envio para WhatsApp

Ao clicar em **Enviar WhatsApp**, é aberta uma conversa com mensagem automática contendo:

- Dados do cliente
- Tipo de crédito
- Valor
- Prazo
- Parcela estimada
